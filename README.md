 Warehouse Inventory System

 Description
This is a simple Java console-based application to manage warehouse inventory using HashMap and ArrayList  
It demonstrates basic Object-Oriented Programming  concepts and Java Collections.



 Features
 Add items  
 Remove items  
 Update item quantity  
 Search items by ID or Name  
 Display all items  
 Input validation for quantity and price  



  Technologies Used
 Java (OOP Concepts)
 HashMap
 ArrayList
 Scanner (User Input)

javac Item.java Inventory.java Warehouse.java
